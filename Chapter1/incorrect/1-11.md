正の整数 x と y の最大公約数とは，x と y の両方を割り切る最大の整数 d のことである. x > y のときに最大公約数 gcd(x, y) を計算するユークリッドのアルゴリズムは，このタスクをより小さい問題に帰着する. gcd(x, y) = gcd(y, x mod y) ユークリッドのアルゴリズムが正しいことを証明せよ． 

x = 36, 23, 13, 10, 3, 1
y = 23, 13, 10, 3 , 1, 0

パート1：停止性
初期条件は x > y でありmod剰余の性質により常に 0 <= r < y の関係が成り立ち、 gcdは必ず 0 で停止する。

パート2：正当性
左から右へ:
「d が x と y の両方を割り切るなら、d は必ず r も割り切る」
なぜなら r は以下の式で表すことができる。
r = x - q * y

-q * y = y の倍数 = dの倍数
x = dの倍数
r は全ての項が d で割り切れるため r も同様に割り切ることができる。

右から左へ:
「d が y と r の両方を割り切るなら、d は必ず x も割り切る」
なぜなら x は以下の式で表すことができる。
x = q * y + r

q * y = yの倍数 = dの倍数
r = dの倍数
x は全ての項が d で割り切れるため x も同様に割り切ることができる。


x mod y を r と置くと gcd(x, y) = gcd(y, r) となる。
以上の条件
        常に 0 <= r < y
        dが {r = x - q * y}, {x = q * y + r}のどちらの式も割り切れることができる。
により、x と y の両方を割り切る最大の整数 d を求めることができる。
