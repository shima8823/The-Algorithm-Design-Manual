Multiply(y, k+1)
y = y, z = k+1
q = (k+1) / c
r = (k+1) mod c

return(Multiply(cy, [z/c]) + y · (z mod c))
= (cy) * q + y * r ※(cy) * qはMultiply(cy, q)であるため
= y * (c*q) + y * r
= y * (c*q + r)

ここでk+1はq,rの定義から除法の定義より
k+1 = (c*q + r)
